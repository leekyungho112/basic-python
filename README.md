# 1.theory

- Data Type
  - a_string = "like this"
  - a_number = 3
  - a_float = 3.12
  - a_none = None (존재하지 않는다)
  - a_boolean = False
- Sequence type(열거형타입)

  - List (mutable)
    - days = ['1','2','3','4','5]
    - len(days)
    - '1' in days
  - Tuples and Dicts(immutable)

    - 변형을 허용하지 않는다
    - ays = ('1','2','3','4','5)

  - dictionary

    - people = {
      "name" : "lee",
      "age": 29
      "korean": True

    }

- Function
  - def 로 정의한 후에 함수명을 작성
  - def hello():
    print("hello")
  - 정의한 함수의 body안에 들여쓰기로 정의한다.
